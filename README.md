# JSB Design System — Step 01

## Overview

This project is a custom design system built for **JSB Business Solutions Group**, a Detroit-based consulting firm specializing in operations, financial readiness, and business growth strategy.

The purpose of this project is to create a clean, consistent, and scalable UI foundation that can support a full production website. The design focuses on clarity, usability, and a professional look aligned with a B2B consulting brand.

---

## Objective

This project was completed as part of a structured developer training program. The goal of Step 01 was to:

- Build reusable UI components
- Define design tokens (colors, typography, spacing)
- Create a consistent layout system
- Improve visual hierarchy and overall design quality

---

## Tech Stack

- Next.js  
- React  
- TypeScript  
- CSS Variables  
- Component-based architecture  

---

## Features

- Design tokens for color, spacing, and typography  
- Reusable components:
  - Buttons  
  - Cards  
  - Forms  
  - Alerts  
  - Tables  
  - Modals  
  - Toasts  
- Structured layout with navigation and sections  
- Smooth scrolling navigation  
- Clean and responsive UI  

---

## Design Approach

This design system focuses on:

- **Clarity** — Simple, readable layouts  
- **Consistency** — Reusable styles across components  
- **Hierarchy** — Clear structure between headings, content, and actions  
- **Trust** — Professional visuals suitable for a consulting business  

The goal was not just to build components, but to start thinking about how design affects user perception and behavior.
